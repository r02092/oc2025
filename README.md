# 2025年度オープンキャンパス作品
- カメラを使って手のひらを撮影し、掌線の長さなどの情報を画像処理によって得て、それに基づいて手相占いをする
- 占いを軽量なローカルLLMでおこない、結果をVOICEVOXを使って四国めたんに読み上げさせる
- できれば手のひらに画像処理や占いの結果などに応じた映像を投影したい
